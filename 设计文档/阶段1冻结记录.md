# 阶段 1 冻结记录

冻结日期：2026-06-17

## 冻结范围

阶段 1 冻结范围为本地 CLI 版本，不包含 FTP/SFTP、Web 平台、DB/Kafka/RESTful 适配器。

已冻结能力：

- 本地文件、目录、ZIP、GZ 输入。
- `mapping_dx.ini` 表映射和字段映射。
- OP 中间表解析。
- TPD JSON 规则聚合。
- Rust 表达式执行：`max`、`count(distinct)`、`CRC64`、`substring`、`locate`、`length`、字符串拼接、环境变量、简单 `case when`。
- 输出包结构：CSV、INI、`load.ctl`、`result.csv`。
- `load.toml` 支持 PostgreSQL 和 ClickHouse。
- `load.toml` 字符串字段支持 `${VAR}` 和 `${VAR:-default}`。
- ClickHouse 表名默认小写，可通过 `clickhouse.table_name_case` 配置为 `upper` 或 `lower`。
- `load.ctl` 在 Unix 平台自动设置为 `755`。

## 回归样本

### NRCELLDU 小样本

命令：

```bash
cargo run --release -- \
  --input fixtures/nrcelldu_sample.csv \
  --config-dir . \
  --output-dir output_stage1_freeze/nrcelldu \
  --collect-id 采集ID \
  --load-type clickhouse \
  --load-config load.toml \
  --encoding UTF-8 \
  --rule-file rules/Tpd_NRCELLDU_q_5g.json
```

结果：

- 输出包：`output_stage1_freeze/nrcelldu/tpd_nrcelldu_q_5g_2026021012/采集ID_202602101200`
- 数据行数：1
- `result.csv.row_count`：1
- `result.csv.success`：1
- `load.ctl` 权限：`755`
- 总耗时：约 0.03 秒

## 输出包验收

输出包文件列表固定为：

```text
load.ctl
result.csv
{table_lower}.csv
{table_lower}.ini
```

验收结果：

- CSV 不包含 header。
- INI 每行一个字段名。
- `result.csv` 包含 `table_name,data_time,row_count,success,collect_time`。
- `result.csv.row_count` 与 CSV 数据行数一致。
- `load.ctl` 自动设置为可执行权限。
- ClickHouse `load.ctl` 默认生成小写表名。
